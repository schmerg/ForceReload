Force Reload Options

ForceReload is a minimal extension that hard reloads the current tab to overcome caching issues when using Developer Tools for Google Chromeâ„¢.

It does this by closing the tab and opening a new one to replace it - this seems to convince the browser to reload resources properly.

See this bug report for examples of how this affects people.

User options

By default, the extension only reacts to the toolbar button, but if you want it will also react to the Ctrl-F5 keystroke thereby over-riding the default reload all action.
